
# 命令行工具
Vue.js 提供一个官方命令行工具，可用于快速搭建大型单页应用。

```shell
# 创建一个基于 webpack 模板的新项目
$ vue init webpack vue-my-project
# 这里需要进行一些配置，默认回车即可
```
This will install Vue 2.x version of the template.

For Vue 1.x use: vue init webpack#1.0 vue-my-project
