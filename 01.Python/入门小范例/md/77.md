#### 77 异位词

两个字符串含有相同字母，但排序不同，简称：互为变位词

```python
from collections import Counter

# 

def anagram(str1, str2):
    return Counter(str1) == Counter(str2)

anagram('eleven+two', 'twelve+one')  # True 这是一对神器的变位词
anagram('eleven', 'twelve')  # False
```
