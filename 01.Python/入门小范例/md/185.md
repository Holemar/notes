### 第185个小例子：重新排序 DataFrame 的列

下面给出 2 种简便的小技巧。先构造数据：

```python
df = pd.DataFrame(np.random.randint(0,20,size=(5,7)) \
,columns=list('ABCDEFG'))
df
```

方法1，直接了当：

```python
df2 = df[["A", "C", "D", "F", "E", "G", "B"]]
df2
```

方法2，也了解下：

```python
cols = df.columns[[0, 2 , 3, 5, 4, 6, 1]]
df3 = df[cols]
df3
```

也能得到方法1的结果。
