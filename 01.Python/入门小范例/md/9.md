#### 9 转为浮点类型　

将一个整数或数值型字符串转换为浮点数

```python
In [1]: float(3)
Out[1]: 3.0
```

```python
In [1]: float('3')
Out[1]: 3.0
```

浮点数最大值
```python
import sys

In[4]: sys.float_info.max                                                      
Out[4]: 1.7976931348623157e+308
```

正无穷大、负无穷大
```python
float('inf') # 正无穷大
float('-inf') # 负无穷大
```

如果不能转化为浮点数，则会报`ValueError`:

```python
In [2]: float('a')
# ValueError: could not convert string to float: 'a'
```
