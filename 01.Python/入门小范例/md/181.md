第 181 个小例子：**rank排名**

rank 方法，生成数值排名，ascending 为False，考试分数越高，排名越靠前：

```python
In [36]: df = pd.DataFrame({'a':[46, 98,99, 60, 43]} )) 
In [53]: df['a'].rank(ascending=False)                   
Out[53]: 
0    4.0
1    2.0
2    1.0
3    3.0
4    5.0
```
