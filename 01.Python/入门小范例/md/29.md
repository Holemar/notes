#### 29  转为字典　　

创建数据字典

```python
# 方法1：使用dict
In [1]: dict()
Out[1]: {}
In [2]: dict(a='a',b='b')
Out[2]: {'a': 'a', 'b': 'b'}

# 方法2：zip
In [3]: dict(zip(['a','b'],[1,2]))
Out[3]: {'a': 1, 'b': 2}

# 方法3：嵌入元组的列表
In [4]: dict([('a',1),('b',2)])
Out[4]: {'a': 1, 'b': 2}

# 方法4：自典型字符串
In [1]: s = "{'a':1, 'b':2}"                                                    
In [2]: eval(s)                                                                 
Out[2]: {'a': 1, 'b': 2}
```
