### 共享变量未绑定之坑
有时想要多个函数共享一个全局变量，但却在某个函数内试图修改它为局部变量：

```python
i = 1
def f():
    i+=1 #NO!
    
def g():
    print(i)

```

应该在f函数内显示声明i为global变量：

```python 
i = 1
def f():
    global i # YES!
    i+=1
```
